User Stories and Work Items week_02

1.User story: As shopper, I want to browse products by category   so that I can find handcrafted items that interest me.  *Work Item: Categories are visible on the homepage and navigation menu.
2.User Story: As a seller, I want to list my handcrafted products with photos and descriptions   * Work Item: Create product listing form with image upload and category selection.
3.User Story: As a seller, I want to manage my inventory and update product availability   *Work Item: Build seller dashboard with inventory management and stock alerts
4.User Story: As a shopper, I want to view product details including photos, description, and price   * Work Item: Create product detail page with image gallery and add to cart button
5.User Story: As a shopper, I want to add items to a shopping cart   * Work Item: Build shopping cart with item management and total price calculation
6.User Story: As a user, I want to write a text review for a product *Work Item: Build review form and display
7.User Story: As a user, I want to view an artisan's story and all their products *Work Item: Create artisan profile page with products list
8.User Story: As a user, I want the site to work on my phone*Work Item: Implement responsive design (mobile-first)
9.User Story: As a user, I want to register/login as either buyer or seller*Work Item: Build authentication (register/login/logout) Google auth
10.User Story: As a user, I want products to load fast *Work Item: Optimize images and implement lazy loading pictures
